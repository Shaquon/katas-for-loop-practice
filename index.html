Katas Practice

What is a Kata?

Kata is structured practice for consistent improvement. Originating in martial arts, kata are structured routines consisting of patterns.

In this set of katas we will be practicing using for loops in Javascript

The structure for the for loop will be consistent and there will be certain patterns that you will notice as you go through each kata. For example the conditions for your for loop in the will be really similar for each kata. You will just need to change you're start conditions, stop condition, and step condition for each.
